# 풀이
## Java
```java
import java.util.Arrays;

class Solution {
    public int solution(int[] array) {
        int answer = 0;
        
        Arrays.sort(array);
        answer = array[array.length/2];
        
        return answer;
    }
}
```

## JavaScript
```javascript
function solution(array) {
    let answer = 0;
    
    const mid = parseInt(array.length/2);
    array.sort((a, b) => a - b);
    answer = array[mid];
    
    return answer;
}
```
